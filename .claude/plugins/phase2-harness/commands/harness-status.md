## /harness-status

Phase 2 에이전트 하네스의 전체 현황을 조회합니다.

### 현황 수집

```bash
cd ~/phase2-agents-harness

# 정의된 에이전트 수
DEFINED=$(find agents-definition -name "*.md" | wc -l)

# 구현된 에이전트 수 (디렉토리 수)
IMPLEMENTED=$(ls agents-impl/ | wc -l)

# 3파일 패턴 준수 에이전트
VALID_PATTERN=0
for agent_dir in agents-impl/*/; do
  count=$(ls "$agent_dir"*.fl 2>/dev/null | wc -l)
  [ "$count" -eq 3 ] && VALID_PATTERN=$((VALID_PATTERN + 1))
done

# 테스트 현황
npm test -- --passWithNoTests 2>&1 | tail -20
```
