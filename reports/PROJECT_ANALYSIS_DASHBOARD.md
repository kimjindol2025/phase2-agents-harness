## 🔴 P0 상세 분석

### 1️⃣ gogs-server-fl2 (순환 의존성)

**분석 결과**:
```
라우팅 모듈
  ↓
  인증 정보 필요 (의존)
  ↑
인증 모듈
  ↓
  라우팅 정보 필요 (의존)
```

**위험**: 한 쪽을 수정하면 다른 쪽이 깨질 수 있음
**해결책**: 세 번째 모듈(router-auth-bridge) 분리
**소요 시간**: 6시간
**난도**: 높음
**시작 시점**: 즉시

### 3️⃣ v9-database (N+1 쿼리)

**문제 코드 패턴**:
```freelang
for item in items {
  var detail = database.query("SELECT * FROM details WHERE id = " + item.id)
  process(detail)
}
```

**결과**: 1,000개 아이템 = 1,001개 쿼리 (100배 느림)

**해결책**: 배치 API + 캐싱
```freelang
var details = database.batch_query(item.ids)  // 1개 쿼리
var cached = cache.get(item.ids)              // 캐시 사용
```

**소요 시간**: 4시간
**난도**: 중간
**기대 효과**: 응답 시간 100배 개선

### 7️⃣ bigwash (암호화 미흡)

**현황**:
- 비밀번호 저장: MD5 (약함)
- 토큰: 없음
- 세션: 평문 쿠키

**해결책**: bcrypt + JWT
```freelang
import crypto

// 비밀번호 저장
var hash = crypto.bcrypt(password, cost=10)

// 토큰 발급
var token = crypto.sign_jwt(user_id, secret)

// 세션 (httpOnly)
set_cookie("session", token, httpOnly=true)
```

**소요 시간**: 4시간
**난도**: 중간
**보안 향상**: 10배
