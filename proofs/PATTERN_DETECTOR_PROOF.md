### 역할

**pattern-detector는 핵심 메타-에이전트**:
1. 조직의 모든 반복 문제를 자동 감지
2. 각 문제의 근본 원인 분석
3. 우선순위별 해결책 제시
4. 다른 에이전트에게 작업 위임

### 통합 방식

```
조직 내 모든 프로젝트
       ↓
pattern-detector 스캔
       ↓
6가지 문제 감지
       ↓
  우선순위 선정
       ↓
에이전트 자동 선택:
  - 입력 검증 → security-scanner
  - 순환 의존 → code-analyzer + architecture-analyzer
  - 기술 부채 → refactor-suggester
  - N+1 쿼리 → performance-profiler
  - 테스트 부재 → test-generator
  - 문서 부족 → document-generator
```
